# Bot Template

Универсальный шаблон Telegram-бота на aiogram 3 с чистой и понятной структурой проекта.
Подходит как стартовая база для разработки ботов под разные задачи.

- aiogram 3
- структура по слоям
- SQLite
- .env конфиг
- стартовые handlers
- главное меню
- база для дальнейшего расширения

```bash

cd bot_project
